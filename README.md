# linker
Web application that simplifies a link like goo.gl, etc.

# heroku
Available on https://linker-test.herokuapp.com/

#Steps to start app locally
```
mvn clean install
java -jar target/dependency/webapp-runner.jar target/*.war
```
Type in browser
```
http://localhost:8080/
```
